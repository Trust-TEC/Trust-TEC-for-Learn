Intel Core are streamlined midrange consumer,
workstation and enthusiast computer central processing units
(CPUs) marketed by Intel Corporation.
These processors displaced the existing mid- to high-end Pentium processors
at the time of their introduction, moving the Pentium to the entry level.
Identical or more capable versions of Core processors are also
sold as Xeon processors for the server and workstation markets.
